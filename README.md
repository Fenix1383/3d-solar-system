* `pygame` для отрисовки симуляции\

## 2. Пользование
### Запуск симуляции
Чтобы запустить симуляцию, вам нужно исполнить в терминале скрипт `main.py`. Это инициализирует программу и покажет окно pygame.

![alt text](/img/image-2.png)

### Изменение переменных
Чтобы изменить *гравитационную константу* или *промежуток времени* в интегрировании, нужно нажать клавиши g или t соответственно. После чего ввести нужные числовые данные, и нажать Enter. Вводимые данные можно увидеть в терминале. Чтобы выставить значение по умолчанию введите 0.
